== Description ==
Detail PopUps provides short code available in post editor.
Using Shortcodes [detail_popups] you can easily add detail information text.

== Installation ==
Unzip plugin file and upload "detail-popups" folder under your "/wp-content/plugins/" directory.

Resulted plugin path should be:
"/wp-content/plugins/detail-popups/**"

Activate the plugin at "Dashboard â€“ Plugins" administration page.
